# Iris_Flower_Classifier
🌸 Iris Flower Classifier is a Machine Learning project that predicts Iris flower species (Setosa, Versicolor, Virginica) using sepal and petal measurements. Built with Python, Scikit-learn, and Streamlit for interactive predictions.
# Iris Flower Classification App

This is a Machine Learning web application built using Streamlit and Scikit-learn.

## Features
- Predict Iris flower species
- Displays flower image
- Shows prediction confidence
- Interactive Streamlit UI

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- Matplotlib

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
